=== WooCommerce cse payment gateway ===
Contributors: Henry Tran

Get payment using your cse payment code on your website.

== Description ==
This plugin allow you to accept payments using cse wallet. User can place order on your site and can pay using cse. Simplest way to collect payment directly into your cse wallet and gain user engagements with easy checkout.

== Installation ==
1. Upload wc-gateway-cse-payment folder to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Goto woocommerce->settings->checkout->CSE PAYMENT
4. Provide required information and enable CSE PAYMENT QR gateway
5. Your change APIKey and Rerct
